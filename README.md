# OTime

OTime is a browser stopwatch!

## Installation

Clone repository OTime.

```bash
git clone https://github.com/whoamali/OTime.git
```

## Get started

### Instal dependencies

```bash
npm i
```

### Run project

```bash
npm run dev # Run project in localhost and port 3000
```

### Export

Run the server to output JSON and CSV

```bash
npm run server
```
